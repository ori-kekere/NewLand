# NewLand

NewLand is a Flask-based social media application that allows users to connect, share content, and interact with each other. It features user authentication, profile management, and various forms of content sharing including text posts, art, and videos.

## Features

- **User Authentication**: Secure login, registration, and logout functionality.
- **User Profiles**: customizable profiles with bios and profile pictures.
- **Social Interaction**:
    - **Follow System**: Follow and unfollow other users.
    - **Likes**: Like posts, art, and videos.
    - **Comments**: Comment on posts, art, and videos.
- **Content Sharing**:
    - **Text Posts**: Share your thoughts with text updates.
    - **Art**: Upload and share your artwork.
    - **Videos**: Share video content.

## Tech Stack

- **Backend**: Python, Flask
- **Database**: SQLAlchemy, SQLite
- **Templating**: Jinja2
- **Authentication**: Flask-Login

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository_url>
    cd NewLand
    ```

2.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Run the application:
    ```bash
    python app.py
    ```

2.  Open your web browser and navigate to:
    `http://127.0.0.1:5000`
